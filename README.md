# Face Recognition Attendance System

## Overview

The Face Recognition Attendance System is an AI-powered attendance management application that automates the process of recording attendance using facial recognition technology. The system detects and recognizes individuals through a webcam and marks their attendance automatically, eliminating the need for manual attendance tracking.

By leveraging Computer Vision and Machine Learning techniques, the application provides a fast, accurate, and efficient attendance management solution for educational institutions and organizations.

---

## Features

* Real-time face detection using webcam
* Face recognition-based attendance marking
* Automatic attendance logging
* Student registration and dataset creation
* Attendance record management
* User-friendly interface
* Fast and accurate identification
* Reduced manual effort and human errors

## Usage

### Step 1: Register Faces

Capture and store images of users for training.

### Step 2: Train the Model

Run the training script or notebook to generate the face recognition model.

### Step 3: Start Attendance System

Launch the attendance application.

```bash
python attendance.py
```

### Step 4: Mark Attendance

The system will:

* Detect faces
* Identify registered users
* Record attendance automatically
